# Pokedex API — Express.js + MongoDB (TP1 → TP4)

API REST type Pokedex en **Node.js / Express**, avec **MongoDB**, authentification **JWT**, gestion des rôles (**USER / ADMIN**), gestion des Pokémon, des régions, et d’un dresseur (Trainer) avec marquage **SEEN / CATCHED**.

## Stack
- Node.js + Express.js
- MongoDB + Mongoose
- Auth: JWT + bcrypt
- Tests: Jest + Supertest
- Doc: Swagger UI (/docs)
